WHY US?

As the society is evolving, we are started relying more on online platforms than a physical one. There are thousands of online platforms been pop-up in last few years, which is good. No doubt online platforms are good but they work well with students who know the internet world in a better way. What happens when a kid joins online platform a continuous guide is required, because the internet world is quite new to him/her and she/he just want to know every pop-up comes in front of his/her eyes. So, we wanted to make something to help those kids by providing home tutor.

Feature 1

As the user get into the site, home page will appear in front of him/her. Which contains About, Contact, Login/Register fields in navigation bar. And on the main page inside the container (Find your teacher) a clickable button is present, by pressing that button user can search the tutor in any specific city.

Feature 2

In Contact field, user can contact with us if any query or help required regarding the site.

Feature 3

User can either login or register as a student or teacher. After registration student will get id and password for login and the teacher will get teacherâ€™s login id and password.

Feature 4

ML base rating and recommendation system will also be present to increase better user experience and better tutors for the students.

Feature 5

The website will give the exact location of tutor/student. And this will be done using google map API for location fetching.
